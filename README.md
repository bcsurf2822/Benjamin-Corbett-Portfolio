## Getting Started

Clone the project

```bash
  git clone https://github.com/bcsurf2822/Benjamin-Corbett-Portfolio.git
```

Install dependencies

```bash
  cd Benjamin-Corbett-Portfolio
  npm install
```

Run the development server

```bash
  npm run dev
```

Open browser

```bash
  http://localhost:3000/
```
